# Caravel

Official website for Caravel, a community for Tunisia's tech-passionate builders.

## Run locally

No build step or package installation is required:

```bash
python3 -m http.server 3000
```

Then open `http://localhost:3000`.

## Deploy to Vercel

Import this repository into Vercel and deploy with the default settings. The site is fully static, so Vercel serves it directly from the repository root.

Founder assets and the downloadable CV live in `assets/`.
